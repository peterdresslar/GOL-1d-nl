Experimental Data

The files in ./data contain table output from BehaviorSpace parameter sweeps on initial-density through the models (2d-Life (base model), 1d-2d-Life, 1d-Life). Each experiment was run in 2 permutations for the toroidal wrapping settings of On and Off. Our initial analysis takes the raw BehaviorSpace tables and processes them to:

  1. Remove the intermediate steps and output resultant dataframes with only final values for each run
  2. Group data by run, allowing for easy processing of statistics for individual initial-density settings.

The resultant statistics will appear below.
